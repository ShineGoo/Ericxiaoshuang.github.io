用github可以做什么有趣的事情?

比如用搭一个小博客、展示页：
「卖萌页」特喜欢 Jacob 的两个卖萌页面： Fat Slides、fat.github.io 的页面、
「展示 slides」@郭家寶 的 BYVoid’s 演示文稿、
「搭建博客」太多，就不列举了；
比如写一些可以公开的小说、剧本、教程和随笔之类，以便长期更新。例如：
「协作写书」progit/progit · GitHub、
「开源小说」Hoborg by JJ、
「环球旅行记录」github.com 的页面；
因为 GitHub 便于版本控制和多人协作，所以多人翻译工作用它维护是非常棒的。例如：
「文档翻译项目」LCTT/TranslateProject · GitHub、
「字幕翻译项目」jkyin/Subtitle · GitHub、
「书籍翻译项目」astaxie/NPWG_zh · GitHub；
多人合作编辑索引、精选集。例如：
「书籍索引」github.com 的页面、
「习题答案」github.com 的页面；
其他工具：
「生成简历」hit9/resume.pdf · GitHub；
变成一个程序员。
